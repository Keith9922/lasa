## 关键纪律

1. **对内/对外语境严格区分**
   - 对内（代码、注释、文档）：可以写"MiniMax API"、"模型名"等
   - 对外（UI 文案、用户帮助）：只说"AI 分析"、"AI 解析"，**绝不出现服务商名字**

2. **沙雕但不低俗**
   - 玩梗 OK，但不要用粗口、人身攻击、暗示性内容
   - 便便插画走"可爱卡通陶土感"，不走"恶心写实"

3. **健康免责声明常驻**
   - 每个结果页底部必须有"仅供娱乐 · 不构成医学建议"
   - 检测到颜色异常（黑、红、灰白）时给软提示而非诊断

4. **本地优先**
   - 不做登录、不做云端存储
   - 最多用 `localStorage` 存最近一次结果方便分享回看
