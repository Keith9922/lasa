# 设计系统：色彩 / 字体 / 间距

## 配色：写实便便色系

### 品牌主色

| 名 | HEX | 用途 |
|----|-----|------|
| `--poo-50`  | `#FFF8E7` | 页面背景（暖米色） |
| `--poo-100` | `#F5E6C8` | 卡片次背景、Tab 容器 |
| `--poo-300` | `#D4A574` | 边框、占位符 |
| `--poo-500` | `#8B5A2B` | **主品牌色** · CTA 按钮、图标、Logo |
| `--poo-700` | `#5C3A1D` | 标题文字 |
| `--accent`  | `#F5C842` | 强调、AI 解析按钮、营养环碳水段 |
| `--hot`     | `#E76F51` | 数字徽章、警告、营养环蛋白段 |

### 便便颜色板（**写实**，非卡通）

> ⚠️ 真实世界的便便颜色都很闷，棕系为主。"绿/红/黑"只是棕色被某种食物色素染了一点点，不会变成纯绿/纯紫/纯黑。

| 名 | HEX | 触发条件 | 视觉印象 |
|----|-----|---------|---------|
| 健康棕 | `#6F4E37` | 默认 | 中等深度的咖啡棕 |
| 深褐 | `#3E2723` | 红肉、可可、铁补充剂多 | 深咖啡 |
| 黄褐 | `#A0834C` | 脂肪偏多 | 闷黄不亮 |
| 灰白 | `#C4B089` | 严重脂肪过多 / 胆汁问题 | 干涸土色 ⚠️ |
| 绿褐 | `#5A5E2E` | 大量绿叶菜 / 转运过快 | 棕底带绿调 |
| 暗红褐 | `#5C3025` | 甜菜、火龙果 | 棕底带红调（不是出血色） |
| 黑褐 | `#1F1410` | 铁剂、蓝莓、甘草 | 暗到几乎看不出，但不是纯黑 ⚠️ |

⚠️ 标注的颜色：在结果页要给"建议留意"软提示。

## 字体

```css
font-family:
  -apple-system,
  "PingFang SC",       /* macOS / iOS 中文 */
  "Hiragino Sans GB",  /* macOS 备选 */
  "Microsoft YaHei",   /* Windows 中文 */
  system-ui,
  sans-serif;
```

不引外部字体（保持轻量 + 中文渲染最自然）。

### 字号梯度

| 用途 | 大小 | 字重 |
|------|------|------|
| Hero 巨标题 | 24px / 1.5rem | 800 |
| 卡片标题 | 20px | 700 |
| 段落 | 14px | 400 |
| 小字 / 次信息 | 12px | 400 |
| 角标 / 标签 | 11px | 600 |
| 极小（水印、时间戳） | 10px | 400 |

### Bristol 数字（结果页主视觉）

| 字号 | 字重 | 颜色 |
|------|------|------|
| 32-40px | 900 (Black) | `--hot` |

## 间距系统（4px 基准）

```
xs  = 4px
sm  = 8px
md  = 12px
lg  = 16px
xl  = 20px
2xl = 24px
3xl = 32px
```

页面 padding：`20px`（移动端默认）。

## 圆角

```
button-pill = 999px (full)
card        = 16px (1rem)
chip        = 999px
input       = 14px
small-tag   = 6px
```

## 阴影

```css
--shadow-sm:   0 2px 4px rgba(0,0,0,.06);
--shadow-md:   0 8px 20px -4px rgba(139,90,43,.5);  /* 棕色调，用在 CTA */
--shadow-lg:   0 20px 40px -8px rgba(0,0,0,.35);    /* 拍立得卡 */
--shadow-card: 0 8px 16px rgba(0,0,0,.08);
```

## 组件令牌（命名规范）

```
.btn-primary    → 棕色实色 CTA
.btn-accent     → 黄色 AI 操作
.btn-ghost      → 透明，仅边框

.chip           → 小圆胶囊标签
.chip-active    → 选中态

.card           → 通用白底卡片
.card-polaroid  → 拍立得风（白边 + 底栏）
.card-result    → 结果主卡（带渐变背景）

.tab-pill       → 圆角 Tab 容器
.tab-pill[aria-selected="true"]  → 选中态
```

## 移动端适配规范

| 视口 | 处理 |
|------|------|
| ≤ 375px | 默认 |
| 376-767px | padding 调到 24px |
| ≥ 768px | 内容居中 max-width: 420px |

- 触控目标最小 `44×44px`
- 文本最小 `12px`，段落正文 ≥ `14px`
- iOS safe-area：顶部 `env(safe-area-inset-top)`、底部 `env(safe-area-inset-bottom)`
