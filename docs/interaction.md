## Tab 详细行为

### Tab 1：快捷选择

- **网格**：3 列 × N 行的食物卡（emoji + 名字）
- **选中态**：背景变色 + 边框加深 + 显示份量小标 `×1 适中` / `×2 暴食`
- **份量交互**：再次点击同一卡片 → 弹出份量调节（少 / 适中 / 多 / 暴食）
- **预设食物**（v0.1）：火锅、烧烤、奶茶、汉堡、寿司、沙拉、披萨、面条、早餐、炸鸡、蛋糕、+ 更多
- **+ 更多**：弹出二级菜单展示完整食物库（来自 [food-macros.json](../research/food-macros.json)）

### Tab 2：描述一下

- **大文本框**（约 6 行高度）+ placeholder 示例
- **占位提示**："例如：中午吃火锅吃了两盘羊肉、一盘青菜，是清汤锅，喝了两杯啤酒……"
- **「✨ AI 解析」按钮**：点击后调 AI（详见 [ai-integration.md](ai-integration.md)）
- **解析中态**：按钮文字变 "AI 解析中…"，加 spinner
- **解析结果卡**：显示在按钮下方
  - 列表展示 `[emoji] [食物名] [份量g]`
  - 每行可点击修改份量
  - 右上角 `AI 估算` 标签
  - 底部 `+ 还吃了别的` 按钮 → 重新激活文本框追加输入

## 今日摄入（共享，跨 Tab）

- **常驻**在 Tab 内容下方
- **数据源**：Tab1 选中的食物 + Tab2 解析出的食物**合并去重**
- **冲突策略**：同名食物，份量取较大值（用户的"再加一点"语义）
- **清空**按钮：二次确认 toast
- **空状态**："还没加东西～挑几个开始吧"

## 主 CTA「💩 开 拉 ！」

- **样式**：圆角矩形大按钮，棕色实色，pulse 缩放动效
- **禁用态**：今日摄入为空时灰显，文案变 "选点东西吧"
- **点击后**：触发出卡动效（详见 [animation.md](animation.md)）
- **副文案**：按钮下方小字"仅供娱乐 · 不构成医学建议"

## 分享交互

- **「📤 分享给朋友」**：调 `html2canvas` 把卡片截图为 PNG → 调用 Web Share API（手机端）/ 复制到剪贴板（桌面端）
- 截图时**临时隐藏**：按钮、动效、调试 UI；只保留卡片本身 + 一行水印 `lasa.app · 拉啥`
- 同时弹出文案模板供复制：`"AI 说我明天会拉出 Bristol Type 6 (糊状)，吐槽：xxx"` + 链接

## 移动端适配要点

| 屏宽 | 处理 |
|------|------|
| ≤ 375px | 默认设计稿尺寸 |
| 375-768px | 同移动布局，仅 padding 调整 |
| ≥ 768px | 内容居中固定宽度 420px，背景填空 |

- **触控**：所有可点元素最小 44×44px
- **fixed 元素**：CTA 不用 fixed（用 sticky 或正常流），避免遮挡 iOS 软键盘
- **safe-area**：底部加 `env(safe-area-inset-bottom)`
