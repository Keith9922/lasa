# 产品定义 v0.1（已锁定）

## 一句话

**告诉我今天吃了啥，我猜你明天拉啥** —— 一个基于"碳水→量、蛋白质→形、脂肪→质"伪科学理论的沙雕预测网站，输出可分享的拍立得💩卡。

## 目标用户

- 18-35 岁，愿意分享沙雕内容的网友
- 主要使用场景：手机端、社交平台、群聊里转发
- 不期待医学准确，期待有梗、好玩、有反差

## MVP 功能范围

### ✅ MVP 必做

1. **食物输入**
   - 双 Tab：快捷选择（食物芯片网格） + 描述一下（自然语言文本）
   - 两 Tab 共享同一个今日摄入，可混用
2. **AI 解析**
   - 文本输入交给 AI 解析为结构化食物清单（含份量克数）
   - 解析结果可在 UI 上手动调整份量
3. **预测引擎**
   - 累加宏量营养素（碳水/纤维/蛋白质/脂肪/卡路里）
   - 输出：Bristol Type 1-7 + 颜色 + 油亮 + 漂浮 + 气味强度 + 体积
4. **结果展示**
   - "马桶接屎 + 拍立得"出卡动效
   - 拍立得卡片：参数化💩可视化 + 数据 chips + AI 吐槽文案
   - 三色环展示碳水/蛋白质/脂肪比例 + 卡路里
5. **分享**
   - 卡片导出为 PNG（html2canvas / dom-to-image）
   - 复制吐槽文案
   - 二维码回链（可选）
6. **健康免责**
   - 全站底部常驻"仅供娱乐"
   - 异常颜色（黑/红/灰白）给软提示

### ❌ MVP 不做

- 用户登录、云端账号
- 历史记录、趋势图（最多本地存最近一次）
- 后端数据库
- 真实热量精确计算（凭 AI + 食物库估算即可）
- 多语言（先做中文）

### 🔮 后续可加

- 历史记录（纯 localStorage）
- 收藏夹 / 神预测榜单
- 朋友 PK 模式
- 微信小程序版

## 用户流程

```
[首页 · 快捷选择 Tab]
    点食物芯片 / 调份量
    ↓
[首页 · 描述一下 Tab（可切换）]
    输入"火锅吃了两盘羊肉..."
    点"AI 解析" → 显示解析结果
    ↓
[共享今日摄入]
    确认所有食物
    ↓ 点 "💩 开 拉 ！"
[出卡动效 · 3.5 秒]
    马桶升起 → 屎掉落 → 水花涟漪 → 拍立得弹出
    ↓
[结果页]
    主卡：参数化💩 + Bristol 数字 + 属性 chips + AI 吐槽
    营养环：碳水/蛋白/脂肪比例 + 卡路里
    （可展开）科学小课堂：为啥是这个结果
    ↓
[分享 / 再来一顿]
    导出图片 / 复制文案 / 重新输入
```

```
用户输入
  │
  ├─ Tab1: FoodChip[]  ─┐

                       PoopPrediction {

                       }

                                  │

```

## 平台 & 技术约束

- **平台**：H5 网页，移动端优先（375px 设计基准），桌面端居中卡片兜底
- **技术栈**：Next.js 15 (App Router) + TypeScript + 原生 CSS（沿用 Stori 项目约定，零运行时样式依赖）
- **AI**：MiniMax API（OpenAI 兼容），Server-side 调用避免 key 暴露
- **图标/插画**：SVG 优先（参数化），最终 Bristol 7 类形象用 AI 生成 + 抠图（详见 [research/image-prompts.md](../research/image-prompts.md)）
- **动效**：纯 CSS animation + transition，不引入 framer-motion（保持轻量）

## 调性

- **沙雕指数**：⭐⭐⭐⭐ / 5（适度沙雕，不下三路）
- **配色**：写实便便色系（棕黄主，闷而不亮）
- **文案**：玩梗、自嘲、网友体；忌严肃说教
- **插画**：可爱卡通陶土风
