# 💩 Poop Predictor — 研究资料

这个文件夹存放"吃啥拉啥"网站开发前期的资料。

## 文件结构

- `theory.md` — 三大营养素（碳水/蛋白质/脂肪）对粪便特征的影响（核心理论）
- `bristol-stool-scale.md` — 布里斯托大便分类法（7 种类型）
- `stool-colors.md` — 大便颜色含义参考
- `food-macros.json` — 常见食物的宏量营养素数据（碳水/蛋白质/脂肪/纤维 per 100g）
- `prediction-rules.md` — 从用户输入 → 预测结果的规则草案
- `sources.md` — 所有引用来源

## 用户原始 idea

> 碳水决定💩的多少，蛋白质决定💩的形状，脂肪决定💩的 ???（用户忘了）

## 研究结论（剧透）

用户的直觉**大致方向是对的**，但科学事实更精确一点：

| 营养素 | 主要影响 | 用户的说法 |
|--------|----------|-----------|
| **碳水（特别是膳食纤维）** | 体积、重量、频率 | ✅ "多少" |
| **蛋白质** | 硬度、气味、颜色（高蛋白+低纤维 → 又硬又臭） | ⚠️ "形状"——其实是硬度，间接影响形状 |
| **脂肪** | **稠度（油腻/漂浮）+ 颜色（浅黄）+ 气味** | 🎯 答案就是这些——脂肪过多 = 油亮、漂浮、浅色（脂肪泻 steatorrhea） |

详情见 `theory.md`。
