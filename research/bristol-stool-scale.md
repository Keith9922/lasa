# 布里斯托大便分类法 (Bristol Stool Scale)

由 Dr. Kenneth Heaton 于 1997 年开发，是医学界标准的大便形态分类工具。共 7 类。

| 类型 | 中文描述 | 英文描述 | 形态 | 健康含义 |
|------|----------|----------|------|----------|
| **Type 1** | 一颗颗硬球（像羊粪/坚果）| Separate hard lumps, like nuts | 🟤🟤🟤 | 严重便秘，停留肠道太久 |
| **Type 2** | 香肠状但表面凹凸 | Sausage-shaped but lumpy | 🥨 | 轻度便秘 |
| **Type 3** | 香肠状但表面有裂痕 | Sausage with cracks on surface | 🌭 | 正常偏干 |
| **Type 4** | 光滑柔软的香肠状（理想）| Smooth, soft sausage / snake | 🐍 | ✅ 最理想 |
| **Type 5** | 软团块、边缘清晰 | Soft blobs with clear-cut edges | 🍪🍪 | 正常偏稀，缺纤维 |
| **Type 6** | 糊状、边缘毛糙的碎块 | Mushy, fluffy pieces with ragged edges | ☁️ | 轻度腹泻 |
| **Type 7** | 完全水样、无固体 | Entirely liquid, no solid pieces | 💧 | 严重腹泻 |

## 用于网站的颜色 / emoji 设计建议

```
Type 1: 深棕色硬球
Type 2: 棕色凹凸香肠
Type 3: 棕色裂纹香肠
Type 4: 中棕色光滑香肠 (绿色对勾)
Type 5: 浅棕色软团
Type 6: 黄棕糊状
Type 7: 浅黄水状
```

## 网站 UI 思路

预测结果时直接展示对应 Bristol Type 的卡通插图 + 数字编号 + 一句话解读。可以做成"扑克牌"风格，让结果更有传播性。
