# 三大营养素 → 粪便特征 映射

## 1. 碳水化合物 / 膳食纤维 → **量、频率、含水量**

- **纤维是大便"体积"的主要决定因素**。纤维通过四种机制影响大便：
  1. 持水能力（让大便更软更大）
  2. 刺激肠道菌群增殖（菌群质量本身就构成相当一部分粪便重量）
  3. 缩短肠道转运时间
  4. 增加产气
- **可溶性纤维**（燕麦、苹果、豆类）→ 在结肠中保水 → 软而成形
- **不可溶性纤维**（麦麸、蔬菜茎、坚果皮）→ 机械刺激肠壁 → 加快转运、增加体积
- **可发酵纤维** → 通过发酵产物（短链脂肪酸）刺激肠蠕动
- 一项 RCT 显示：补充纤维平均每周增加 1.19 次排便
- **未消化吸收的碳水**（如乳糖不耐、果糖吸收不良、木糖醇/山梨醇）→ 渗透性腹泻 + 产气 + 腹胀

**总结**：碳水决定**量**——✅ 用户说对了。

---

## 2. 蛋白质 → **硬度、气味、颜色**（不直接决定形状）

- 蛋白质本身**不会**直接导致便秘，但**高蛋白饮食通常伴随低纤维** → 大便变硬、变干、变少
- 一项 AJCN 研究：高蛋白低碳水饮食的人群，**排便频率显著降低，大便硬度显著增加**
- 肾脏处理蛋白代谢需要更多水 → 进一步抽走粪便中的水分 → 更硬
- **气味**：未被小肠完全吸收的蛋白质会被结肠细菌发酵，产生：
  - 含硫氨基酸 → 硫化氢（臭鸡蛋味）
  - 胺类、吲哚、酚类、氨 → 难闻气味
- **红肉、禽肉**最容易产生重口味气味
- 高蛋白可能导致**淡黄色油腻大便**（间接：低纤维 → 胆汁流量减少 → 脂肪吸收下降）

**总结**：蛋白质主要影响**硬度（→ 形状偏向"硬球"）+ 气味**——⚠️ 用户说"形状"算半对，更准确是"硬度"。

## 3. 脂肪 → **稠度（油腻/漂浮）、颜色、气味** 🎯

> **这就是用户忘了的那个！**

- 未被小肠吸收的脂肪进入结肠 → **脂肪泻（steatorrhea）**：
  - **油腻、滑亮、有油花**
  - **漂浮**在水面上（脂肪比水轻 + 产气）
  - **颜色变浅**（淡黄、土黄、灰白色）
  - **极臭**
  - **稀溏、量大**
- 高饱和脂肪饮食的研究对象，反而更容易出现**硬便**（吸收较好时）
- 急性大量摄入脂肪（比如一顿吃太多油腻火锅、奶油蛋糕、椰子油、棕榈油、坚果）→ 短期脂肪泻
- 慢性脂肪泻可能提示胰腺功能不全、胆汁分泌问题、乳糜泻等疾病（→ 网站需要免责声明）

**总结**：脂肪决定**油腻程度 + 颜色 + 漂浮性 + 气味**——这就是用户忘了的答案。

## 综合速查表（用于预测引擎）

| 输入特征 | 推断输出 |
|---------|---------|
| 高纤维（>25g/天）| Bristol 4-5，量大、易排出 |
| 低纤维 + 高蛋白 | Bristol 1-2，硬球、量少、味重 |
| 高脂肪（>40% 热量来源）| Bristol 5-7，油腻、漂浮、浅色 |
| 高糖 + 低纤维 | Bristol 6，稀软（渗透性） |
| 大量乳制品（乳糖不耐人群）| Bristol 6-7，胀气、稀 |
| 红肉 + 啤酒 | 颜色深、气味重、Bristol 3-4 |
| 大量绿叶菜 | 颜色偏绿、Bristol 4-5 |
| 大量胡萝卜/南瓜/红薯 | 偏黄橙色 |
| 大量蓝莓/甜菜 | 偏暗红/紫黑（注意别误判为出血）|
